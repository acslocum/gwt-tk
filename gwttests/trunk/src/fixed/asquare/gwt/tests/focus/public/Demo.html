Programmatic focus bug demo

asquare.gwt.tests.focus.Demo

Description

Programmatically focusing CheckBox fails

GWT Version

Windows 1.1

Platforms Tested

IE6 (XP), FFox 1.5 (XP)

Platforms Affected

All

Reproduction

Click the "focus" button next to the checkbox. 
Expected: "focus" button loses focus, checkbox gains focus
Actual: nothing happens

Detail

The GWT CheckBox component is comopsite of a SPAN, INPUT and LABEL elements. HasFocus.setFocus() is attempting to focus the span.

CheckBox is focusable by normal tabbing.

This bug interferes with implementation of programmatic focus cycles.

Also affects RadioButton.
