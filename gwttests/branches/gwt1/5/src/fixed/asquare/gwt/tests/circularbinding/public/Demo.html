Description

GWT compiler produces infinite loop as a result of super method call in implementation class.

GWT Version

1.2.11, 1.1.10

Platforms Tested

IE6 (XP), FF1.5 (XP)

Platforms Affected

IE6 (XP), FF1.5 (XP)

Reproduction

	Click on the element labeled "Click me"
	
	Expected: text is written to the TextArea
	
	Actual Result: stack overflow error
