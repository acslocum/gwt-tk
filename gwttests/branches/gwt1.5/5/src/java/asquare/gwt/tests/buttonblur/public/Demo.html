Button Blur Bug Demo

asquare.gwt.tests.buttonblur.Demo

Description

onblur event handler is called in Opera whenever a button is clicked

GWT Version

1.3.3

Platforms Tested

Hosted Browser (XP), IE6 (XP), FFox 1.5 (XP), Opera 9.01 (XP), Safari 2.0.4 (OS 10.4.7), Camino 1.0.2 (OS 10.4.7)

Platforms Affected

Opera 9.01 (XP)

Reproduction

click on the buttons below and observe the output in the debug console
	
	Expected: onfocus is reported when button is first clicked, onblur is reported when button is blurred
	
	Actual Result: onblur is reported whenever button is clicked (and again when button is blurred)
